Resumo sobre a disciplina



Arquitetura Cliente-Servidor: A disciplina começa com uma introdução à arquitetura cliente-servidor, que é a base para a maioria das aplicações web. Ela explora como os clientes (navegadores ou aplicativos móveis) se comunicam com os servidores web para solicitar recursos e serviços.

Linguagens de Programação Web: A disciplina aborda as linguagens de programação web fundamentais, como HTML e CSS. Essas linguagens são usadas para criar a estrutura, o estilo e a interatividade das páginas web.

Desenvolvimento do Lado do Servidor: Também foi abordado o desenvolvimento de aplicativos no lado do servidor, como é o caso do PHP.

APIs e Integração de Serviços: A disciplina pode abordar o uso de APIs (Application Programming Interfaces) para integrar serviços de terceiros em aplicativos web.
